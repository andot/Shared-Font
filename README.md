# Shared-Font
A Nintendo 3DS Shared Font

适合3ds模拟器使用的中日韩美欧全区字库

# 14px
- 字库规格：点阵字形，14px

- 字符范围：日版字库的全部字符 + 简中字库的全部字符 + 繁中字库的全部字符 + 韩版字库的全部字符

- 适用环境：3ds模拟器使用各种语言的 ROM 不用切换字库了

- 字库缺陷：点阵字库，字体笔画比较细，低倍分辨率下字体比较模糊，高倍分辨率下还是挺清楚的。
